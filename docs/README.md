# Dokumentation - Sajtstudio.se

**All projektdokumentation finns här.**

## Snabbguide

### För utvecklare

1. **Börja här:** `SITE_STRUCTURE.md` - Komplett översikt över projektet
2. **Mobil/Desktop:** `MOBILE_DESKTOP_INDEX.md` - Hitta mobil vs desktop kod
3. **Responsiv design:** `RESPONSIVE_DESIGN_GUIDE.md` - Best practices

### För AI-agenter

- Se `.cursorrules` i projektroten för instruktioner
- Alla dokument refereras från `.cursorrules`
- Använd `MOBILE_DESKTOP_INDEX.md` när du behöver hitta mobil/desktop-kod

## Uppdatera dokumentation

När du gör ändringar, uppdatera relevant dokumentation:

- Ny komponent → `SITE_STRUCTURE.md`
- Mobil/desktop-ändringar → `MOBILE_DESKTOP_INDEX.md`
- Responsive patterns → `RESPONSIVE_DESIGN_GUIDE.md`
- Strukturändringar → `.cursorrules` (i projektroten)
