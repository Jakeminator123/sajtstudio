SAJTSTUDIO.SE - WEBBPLATS OCH DESIGN BESKRIVNING
==================================================

ÖVERSIKT
--------
Sajtstudio.se är en modern företagswebbplats byggd med Next.js 16, React 19, TypeScript, Tailwind CSS och Framer Motion.
Webbplatsen följer en monokrom minimalistisk designfilosofi med strategiskt placerade accentfärger och stora, kraftfulla typografiska element.

WEBBPLATSENS SYFTE
------------------
Sajtstudio är ett webbdesignföretag som skapar skräddarsydda, toppmoderna hemsidor för utvalda företag.
Webbplatsen fungerar som både portfolio och marknadsföringsmaterial för att visa företagets kompetens och attrahera nya kunder.

SIDSTRUKTUR
-----------
- Startsida (/): Hero-sektion, Om oss, Unique Selling Points, Tjänster, Portfolio-animationer,
  Tech showcase (interaktiv Pacman-demo), Scroll-illusioner, Process, Omdömen, Stor CTA-sektion
- Portfolio (/portfolio): Portfolio-galleri med projekt
- Kontakt (/contact): Kontaktformulär
- Sajtgranskning (/sajtgranskning): Website analyzer-funktionalitet
- Utvärdera (/utvardera): Utvärderingsverktyg

DESIGNFILOSOFI
--------------
Designen följer en monokrom minimalistisk stil med:
- Svart och vitt som primära färger
- Strategiskt använda accentfärger för framhävdning
- Stora, kraftfulla typografiska element
- Generös whitespace för luftig känsla
- Smooth animationer och övergångar
- Moderna interaktiva element

ANIMATIONER
-----------

VARAKTIGHET:
- Instant: 0.1s
- Fast: 0.25s
- Normal: 0.4s
- Slow: 0.8s
- Slower: 1.2s
- Slowest: 2s

EASING (Rörelsefunktioner):
- Default: cubic-bezier(0.25, 0.1, 0.25, 1) - Smooth standard
- EaseIn: cubic-bezier(0.42, 0, 1, 1)
- EaseOut: cubic-bezier(0, 0, 0.58, 1)
- EaseInOut: cubic-bezier(0.42, 0, 0.58, 1)
- Spring: cubic-bezier(0.34, 1.56, 0.64, 1) - Bounce-effekt
- Smooth: cubic-bezier(0.4, 0, 0.2, 1) - Extra smooth

ANIMATIONSTYPER:
- FadeIn: Opacity 0 → 1
- SlideUp: Opacity 0 + translateY(30px) → 1 + translateY(0)
- SlideIn: Opacity 0 + translateX(-30px) → 1 + translateX(0)
- Scale: Opacity 0 + scale(0.9) → 1 + scale(1)

SPECIELLA EFFEKTER:
- Shimmer: Gradient-animation på länkar och knappar (blå → röd → blå)
- Glow: Box-shadow med accentfärger för framhävdning
- Shake: Skak-animation för CTA-knappar vid interaktion
- Rain: Regn-effekt i hero-sektionen
- Lightning: Åsk-effekt i hero-sektionen
- Magnetic: Magnetisk hover-effekt på knappar

KOMPONENTER OCH DESIGNPATTERNS
-------------------------------

HEADER/NAVIGATION:
- Fast positionerad header som ändrar stil vid scroll
- Desktop: Logo (vänster) → Navigation links (mitten) → CTA-knapp (höger)
- Mobil: Logo (vänster) → Hamburger-meny (höger)
- Shimmer-effekt på navigation-länkar vid hover
- CTA-knapp med gradient (blå → röd), shimmer och glow-effekt

HERO-SEKTION:
- Fullskärms-sektion med video-bakgrund
- Stora, animerade rubriker med word reveal-effekt
- Regn och åsk-effekter för atmosfär
- Magnetiska CTA-knappar med interaktiv hover-effekt
- Scroll-indikator som försvinner vid scroll

KNAPPAR (BUTTONS):
- Primary: Svart bakgrund, vit text, hover till accent
- Secondary: Accent bakgrund, vit text
- Outline: Border, hover fyller bakgrund
- CTA: Gradient (accent → tertiary), shimmer-effekt, glow-effekt, rounded-full

MODALER:
- Mörk overlay (rgba(0, 0, 0, 0.6))
- Ljus modal med subtil skugga
- Smooth fade-in animation
- Custom scrollbar för långt innehåll

KORT (CARDS):
- Subtila skuggor
- Hover-effekter med scale och glow
- 3D-transform-effekter på vissa kort

FORMULÄR:
- Minimalistisk design
- Accent-färgad fokus-indikator
- Tydlig feedback vid validering

BAKGRUNDSEFFEKTER
-----------------
- Video-bakgrunder med grayscale-filter och låg opacity
- Animerade GIF-bakgrunder med slow motion-effekt
- Gradient-bakgrunder för vissa sektioner
- Partikel-effekter i CTA-sektioner
- Matrix-stil bakgrunder för vissa sektioner

INTERAKTIVA ELEMENT
-------------------
- Pacman-spel i tech showcase-sektionen (visar tech vs design)
- Scroll-baserade parallax-effekter
- 3D-transform-effekter på kort
- Custom cursor-effekter (på vissa sektioner)
- Magnetiska knappar som följer muspekaren

TILLGÄNGLIGHET
--------------
- Tydliga fokus-indikatorer (blå outline)
- Skip-länk för keyboard-navigation
- Screen reader-vänligt innehåll
- Reduced motion-stöd för användare som föredrar mindre animationer
- Touch-friendly storlekar på interaktiva element (min 44x44px)

PERFORMANCE OPTIMERINGAR
------------------------
- Lazy loading av sektioner som visas senare
- Prefetching av länkar vid hover
- Prefetching av komponenter när användaren scrollar nära dem
- Content visibility optimization för bättre scroll-prestanda
- Optimerade bilder och videor
- Smooth scrolling med touch momentum

SAMMANFATTNING
--------------
Sajtstudio.se använder en monokrom minimalistisk design med svart och vitt som grund, kompletterad med en kraftfull blå accentfärg (#0066FF)
och sparsamt använd röd (#FF0033) för sekundära accentpunkter. Designen kännetecknas av stora, kraftfulla typografiska element,
generös whitespace, smooth animationer och moderna interaktiva effekter. Webbplatsen är byggd med fokus på prestanda, tillgänglighet
och en premium användarupplevelse.
